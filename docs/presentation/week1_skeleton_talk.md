# 第1周项目骨架讲解稿

各位老师好，我当前完成的是一个面向 RAG 工程学习的 FastAPI 后端骨架项目。

这一阶段的目标不是直接完成完整 RAG 系统，而是先把后续开发所需的基础工程结构搭起来，包括服务入口、配置管理、日志、测试和项目文档。

目前已经完成了以下内容：

1. 搭建 FastAPI 最小服务骨架
2. 实现了 `/health` 和 `/version` 两个基础接口
3. 使用 `.env` 和 `config.py` 完成基础配置读取
4. 使用 `logging_config.py` 实现控制台和文件日志输出
5. 使用 pytest 为 `/health` 编写了首个自动化测试
6. 完成了 README、周志、论文资料入口和评测占位文件

这一阶段最重要的成果不是功能多少，而是项目已经具备了：
- 可启动
- 可测试
- 可记录日志
- 可持续迭代
- 可进入下一周文档导入开发

下一步我会在 `develop` 分支继续实现 TXT 和 PDF 文档导入，为后续切块和检索做准备。
