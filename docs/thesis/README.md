# Thesis Docs（论文资料入口）

## 1. 这个目录当前的作用

这个目录用于收集本项目后续论文/毕业设计相关资料，当前阶段先建立入口，不展开正式论文撰写。

当前目标：
- 收集学校论文/毕设时间表
- 收集论文格式要求
- 记录项目与论文相关的技术决策
- 为后续开题、正文、答辩材料做准备

---

## 2. 当前项目与论文的关系

当前项目是一个面向 RAG 工程学习的 FastAPI 后端骨架项目。  
第 1 周主要完成了项目初始化、后端骨架、配置、日志、测试和 README。

这些内容后续可以作为论文中的以下素材来源：

- 系统设计与实现
- 后端架构说明
- 配置管理方案
- 日志与测试设计
- 项目阶段推进记录

---

## 3. 当前已完成的论文准备动作

- 建立 `docs/thesis/` 目录
- 将论文相关资料整理入口放入项目
- 明确后续需要收集的内容
